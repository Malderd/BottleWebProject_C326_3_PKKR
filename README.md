# Веб-приложение «Graphora»

## О проекте

**Graphora** — веб-приложение, разработанное на основе фреймворка Bottle. Проект представляет собой систему для автоматизации вычислений на графовых структурах с использованием специализированных алгоритмов теории графов.

## Возможности

- Поиск Эйлерова цикла или цепи графа с использованием алгоритма Иерхольцера
- Поиск Гамильтонова цикла или цепи графа методом полного перебора с возвратом
- Выявление сообществ графа перебором всех подмножеств вершин
- Разбиение графа на компоненты сильной связности с использованием алгоритма Косарайю

Все результаты работы алгоритмов можно сохранить в текстовый файл, выбранный пользователем.

## Установка
Ниже приведена последовательность действий для запуска проекта на локальной машине.

### 1. Клонировать репозиторий

```bash
git clone <ссылка_на_репозиторий>
cd BottleWebProject_C326_3_PKKR
```

### 2. Создать виртуальное окружение

```bash
python -m venv venv
```

Активация (Windows):

```bash
venv\Scripts\activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустить приложение

```bash
python app.py
```

После запуска открыть в браузере:

```
http://localhost:5555/
```
